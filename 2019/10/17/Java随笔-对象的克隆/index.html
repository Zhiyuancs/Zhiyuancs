Java随笔--对象的克隆

2019-10-17
Java

java的引用类型（包括类、接口、数组等复杂类型）是无法通过等号直接赋值的。

如果要实现对象的复制，需要重写clone方法，所有类都继承自Object类，该类有一个protected的clone方法。
